## 📝 고객 요구사항 정리
- 웹페이지 기본으로 하되 모바일에서도 화면이 잘리지 않고 잘 보이게
- 상품 목록에서 상품 사진 위주로 잘 보이게 끔
- 장바구니에 상품을 추가 할 때, 헤더에서 실시간으로 상품 개수 표시
- 작업 결과물 확인 위한 테스트 url 제공

## 📋 기능 목록
- 화면 설정
  - `<App>`
- 상품 리스트
- 상품을 2개의 컬럼으로 보여준다.
  - `<ProductList />`
- 상품 설명
- 상품의 사진이 50% 제목, 설명, 가격, 담기버튼이 나머지 50%
  - `<ProductItem>`
- 담기버튼
- 담기버튼 누르면 담김! 버튼으로 바뀜
  - `<CartButton>`
- 장바구니
- 장바구니에 상품이 몇 개가 담겼는지 숫자로 보여준다.
  - `<Header>`
